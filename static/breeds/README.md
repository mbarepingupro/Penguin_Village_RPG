# Penguin Breed Sprites
Alternate penguin base sprites unlocked through prestige and achievements.

Dimensions: same as penguin.png
Format: PNG with transparent background
No anti-aliasing — pixel art only

Required files:
- classic_black.png (default — copy of penguin.png)
- arctic_white.png (prestige 1)
- royal_blue.png (prestige 2)
- golden_emperor.png (prestige 3)
- shadow_purple.png (achievement: defeat 100 monsters)
- frost_crystal.png (achievement: reach level 30)
- neon_pink.png (Twitch sub exclusive)
