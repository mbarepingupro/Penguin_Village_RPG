# Item Icons
Square icons shown in inventory, shop, and tooltips.

Dimensions: 32x32 pixels
Format: PNG with transparent background
No anti-aliasing — pixel art only

Subfolders:
- hats/ — hat item icons (baseball_cap.png, beanie.png, etc.)
- outfits/ — outfit/cape icons (hoodie.png, tuxedo.png, etc.)
- footwear/ — shoe icons (sneakers.png, sandals.png, etc.)
- accessories/ — accessory icons (sunglasses.png, dragon_wings.png, etc.)

Naming convention: use the item id from BOUTIQUE_ITEMS in app.py
Example: baseball_cap.png, viking_helmet.png, dragon_wings.png
