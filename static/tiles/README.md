# Tile Sprites
Isometric ground tiles for the village map.

Dimensions: 64x32 pixels (isometric diamond)
Format: PNG with transparent background
No anti-aliasing — pixel art only

Required files:
- snow.png — tile type 0 (walkable, default ground)
- path.png — tile type 1 (walkable, dirt road)
- water.png — tile type 2 (not walkable, frozen pond/ice)
- tree.png — tile type 3 (not walkable, tree on base tile)
- fence.png — tile type 5 (not walkable, fence/wall)
- building_zone.png — tile type 4 (optional, hidden under buildings)
