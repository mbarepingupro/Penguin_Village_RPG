# Penguin Wearing Sprites
Items as they appear ON the penguin sprite. Layered on top of the base penguin.

Dimensions: same as penguin.png base sprite
Format: PNG with transparent background — ONLY the item, no penguin
No anti-aliasing — pixel art only

How to create:
1. Open penguin.png as a reference layer in your editor
2. Draw the item on a new layer, positioned correctly on the penguin
3. Hide the penguin layer
4. Export only the item layer

Subfolders:
- hats/ — positioned on top of penguin head
- outfits/ — positioned on penguin body
- footwear/ — positioned at penguin feet
- accessories/ — positioned on penguin face/side

Naming: must match the files in static/items/ exactly
