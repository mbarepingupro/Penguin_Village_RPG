# Building Sprites
Isometric building sprites for the village map.

Format: PNG with transparent background
No anti-aliasing — pixel art only

Sizes by footprint:
- 3x3 buildings: 192x160 pixels (width x height including 3D)
- 3x2 buildings: 192x128 pixels
- 2x2 buildings: 128x96 pixels

- hotel.png (3x3)
- sea_lion_pit.png (3x3)
- club_soda.png (3x2)
